## 판례
기본구조 : 법원명 + 선고일자 + 사건번호(number1, number2, number3) + 판결유형

### ex) 대법원 2015. 2. 26.선고 2010다106436 판결

| court | date | number1 | number2 | number3 | type | 참조정보 |
| ----- | ---- | ------- | ------- | ------- | ---- | -------- |
| 대법원 | 2015. 2. 26. | 2010 | 다 | 106436 | 판결 | 대법원 2015. 2. 26.선고 2010다106436 판결 |

- 필수값 : 법원명 + 사건번호
- 선고일자 없으면 ‘법원명 + 사건번호’로만 작업

### ex) 대법원 2010다106436 판결

### 기타
약어
 - 대결->대법원 결정
 - 대판->대법원 판결
 - 대판(전)->대법원 전원합의체판결

- 판례링크는 사건번호만 사건명(사건번호) 일경우 사건번호만 밑줄
- 대법원 2000.11.11 선고 2000두12354 판결 (확정) -> (확정)은 링크범위 아님

## 문헌

• 문헌 검색 -> 구분 값이 ‘기사’인 경우에만 링크 걸어줌, 외국 문헌도 포함됨 (‘국내/외국’ 문헌 모두 검색대상)

(문헌 검색하여 나왔는데 구분값이 ‘단행’일 경우에 링크 제외)
- 단행 -> 단행본, 사법행정간행물
- 기사 -> 논문, 평석 등 단행(단행본, 사법행정간행물)을 제외한 나머지 모두 기사로 보면 됨.

- 본문에 없어도 참고문헌에 나와있는 문헌은 모두 링크작업
- 같은 문헌이라도 책 페이지 수가 다르면 엑셀에 추가해서 작업

(단, 엑셀에 들어가는 참조정보의 문헌정보는 동일하게 들어감)

ex. 
- 주1) 최주현, 대법원판례해설(1), 250면 -> 참조정보 : 최주현, 대법원판례해설(1), 250면
- 주2) 최주현, 주(1), 8면 -> 참조정보 : 최주현, 대법원판례해설(1), 8면


======

- 문헌작업하는것-논문, 평설, 간행물,사법행정간행물(단행본은 작업안함)
- 단행, 사법행정간행물 -> 경우에만 구분값 '단행'
- 나머지(논문, 평석 등) -> 구분값 '기사'
- 외국도서도 검색해볼것.
- 본문에 없어도 참고문헌에 나와있는 문헌은 다 링크작업
- 같은 문헌이라도 책 페이지 수가 다르면 엑셀에 추가해서 작업
- 문헌참조 링크 시 -> 링크 범위 : '~이하' 언급시 '~이하'까지 링크 범위O
- 단, '~참조' ,'~참고' 등은 링크 범위X
- 원문에 있는 면수 정보까지 링크 범위
- 졸고(자기나 자기와 관련된 사람의 원고를 겸손하게 이르는 말)는 검색시 저자명에 논문 저자명입력
- 검색시 [법원도서관](https://library.scourt.go.kr/search/total/totalDetailSearch), [종합법률정보](https://glaw.scourt.go.kr/wsjo/mhn/sjo200.do) 참고
- 본문내 기사/단행 구분과 검색결과 기사/단행 구분이 다를경우 검색결과를 우선시함  
## 법령
법령은 PDF내 링크 작업을 하지 않음 

법령
- 법, 시행령, 시행규칙, 규칙, ~법률 에 해당하는것만 작업(‘법령명’은 풀네임으로)
- 줄임말의 법령명이 언급될 경우 -> ‘[국가법령정보센터](http://www.law.go.kr/)’에 검색하여 나오는 풀네임으로)
  
ex) 무슨 법 + 조, 항, 호, 목

필수값 : **법령명, 조**

(작업제외-부칙, 대법원 규칙, 예규, 조례, 지침, 규정)

### ex) ‘민소법 제2조 제1항 제2호 가목’ 일 경우 

| statute | clause | item | ho | mok |
| ------- | ------ | ---- | -- | --- |
| 민사소송법 | 2 | 1 | 2 | 가 |

### ex) ‘민소법 제2조의3 제1항 제2호 가목’ 일 경우 
| statute | clause | item | ho | mok |
| ------- | ------ | ---- | -- | --- |
| 민사소송법 | 2조의3 | 1 | 2 | 가 |

### ex) ‘민소법 제4조 3항’ 일 경우 

| statute | clause | item |
| ------- | ------ | ---- |
| 민사소송법 | 4 | 3 |

### ex) ‘민소법 제4조 3항’이 언급된 이후, 바로 ‘2항’ 이 언급될 경우
(항, 호, 목 중 1개의 정보만 언급될 경우에 원문에서 앞내용 파악하여 ‘법령명’과 ‘*조’ 값을 확인하여 작업해줌)

| statute | clause | item |
| ------- | ------ | ---- |
| 민사소송법 | 4 | 2 |

### ex) 민법 196조, 199조 명시될 경우
민법 196조, 민법 199조 2가지 모두 각각 작업

| statute | clause |
| ------- | ------ |
| 민법 | 196 |
| 민법 | 199 |

### ex) 형소법 24①ⅰ -> 형사소송법 24조 1항 1호
| statute | clause | item | ho |
| ------- | ------ | ---- | -- |
| 형사소송법 | 24 | 1 | 1 |

### ex) 구 민법 25조 1항 
구 민법 -> 구 민법

‘구 법’은 구를 명시하고 법명으로 작업 

| statute | clause | item |
| ------- | ------ | ---- |
| 구 민법 | 25 | 1 |

### 기타
 - 가사소송법 제2조 제1항 제2호 가목 21" 일 경우 목항에 가목21로 입력
 - 00법 196조~199조-> 00법 196조, 00법 199조 2가지만 작업
 - 법24①ⅰ -> 법24조 1항 1호
 - 규정이 붙는경우 조 항이 있어도 작업 대상 아님님
 - 외국 법령의 경우 작업대상 아님
## 목차 
원문내용상 제목, 목차 내용에 법령/판례/문헌 링크대상 있는 경우, 작업해야함.


## 입력기

pdf 파일이 읽기전용으로 열려서 작업 안될때

메뉴->편집->맨 밑에 기본설정->가운데 부분에 보면 PDF/A 문서보기를 안함으로 설정해줄것

## 기타
세 항목(판례/법령/문헌) 중에 한개라도 링크정보 없는 경우

-> 서지정보 동일하게 적용하여 채우기

서지정보 논제명까지 동일하나, 참조구분값은 다름
 - 판례 -> 참조판례
 - 법령 -> 참조법령
 - 문헌 -> 참조문헌
(단, 참조구분값만 해당시트에 맞게 적용하여 채우기)

PDF파일내 링크를 생성해야 하는 문구가 여러줄에 나눠져 있을경우 각각의 줄을 각각로 작업


• 서지값 필수정보 누락있으면 안됨(아래 표가 서지 필수정보)

| 순번 | 파일명 (PDF) | 파일명 (PDFA) | INDEX 정보 | 구분 | 제어번호 | 저자 | 논제명 | 참조구분 |
| ---- | ------------ | ------------- | ---------- | ---- | -------- | ---- | ------ | -------- |
| 1 |  |   |  |   |   |   |   |   |


• 서지정보 입력시 구분값 제대로 확인하여 작업
 - 구분값은 2개 중 하나에 해당(기사, 단행)
  * 단행 -> 단행본, 사법행정간행물
  * 기사 -> 논문, 평석 등 
  * 단행(단행본, 사법행정간행물)을 제외한 나머지 모두 기사로 보면 됨.
- 서지 정보에 제목 저자명등이 한자로 되어있을경우 한자로 입력
